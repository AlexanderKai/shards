# Shards

ETS tables on steroids!


## Build

    $ git clone https://github.com/cabol/shards.git
    $ cd shards
    $ make


## Running Tests

    $ make tests


## Building Edoc

    $ make doc

> **Note:** Once you run previous command, a new folder `doc` is created, and you'll have a pretty nice HTML documentation.
